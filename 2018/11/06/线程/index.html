pthread_create

线程的创建是通过pthread_create来完成

新创建的线程是从start_routine开始run的，arg是该函数的参数. pthread_create的用法比较简单，但是需要注意：

  1. 线程创建时 新线程与调用线程的运行顺序不确定
  2. pthread_create函数在调用失败时会返回错误码，但不会像其他PISIX函数设置errno，每个线程都有errno副本。

Thread Termination

thread退出有3种方式：

  1. return from start routine
  2. pthread_exit()
  3. pthread_cancel by other threads.

Note: pthread_create and pthread_exit的void*可以传递更复杂的内容。但是需要注意，如果你传递指针的话，就会存在线程间的race condition，即这个指针指向的对象在1个thread发生变化，在另外1个thread也会做对应的改变。

  • 打印颜色
    echo -e "\e[1;32m This is red text \e[0m"
    \e[1;32m将颜色设置为红色。\e[0m将颜色恢复
    经常使用的颜色有： 重置=0, 黑色=40, 红色=41, 绿色=42, 黄色=43, 蓝色=44, 洋红=45, 青色=46, 白色=47

重定向

ls + . > output 2>&1
如上是从右往左运算的，先将2与1绑定起来，然后将输出重定向到output文件

如果想将输出既输出到stdin，也存1份到output文件，应该用tee

注意tee只能从stdin读取
如果想stderr也输出，就需要重定向绑定
ls + . 2>&1 | tee output

还可以通过cmd < file 来读取文件内容
